# tlias 企业员工管理系统

## 项目简介
基于 Spring Boot + MyBatis + MySQL 的企业员工信息管理系统，实现部门管理、员工管理、登录认证、文件上传等核心功能。

## 技术栈
- Spring Boot
- Spring MVC
- MyBatis
- MySQL
- JWT（登录认证）

## 主要功能
- 部门管理：增删改查
- 员工管理：增删改查 + 多条件分页查询
- 登录认证：JWT + 拦截器
- 文件上传：员工头像上传

## 如何运行
1. 导入 SQL 文件到 MySQL
2. 修改 application.yml 中的数据库连接信息
3. 运行 TliasApplication.java
